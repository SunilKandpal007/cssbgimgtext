how to add an image in the background of a text

For this we use following CSS properites:

  1. background-clip:text "The background is painted within (clipped to) the foreground text."
  2. -webkit-background-clip:text "Determines the background painting area, which determines the area within which the background is painted."
  3. color:transparent "Make color transparent of the text, so that applied background image can be view."
